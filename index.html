Sherpa - simple web api's for web apps

  • Simple for backend developers: Just export functions you wrote using one of the small libraries.
  • Simple for frontend developers: Just call a function and get a promise back, see sherpa.js.
  • Simple for all developers: API's are self-describing and self-documenting, see API documentation.

▸ Show example of exporting a sherpa function in Go

package main;

import (
    "net/http"
    "bitbucket.org/mjl/sherpa"
)

var count int64
func RequestCount() int64 {
    count += 1
    return count
}

func main() {
    functions := map[string]interface{}{
        "requestCount": RequestCount,
    }
    api, _ := sherpa.NewHandler("http://localhost:8080/example/",
        "example", "Example API", "0.0.1", functions)
    http.Handle("/example/", http.StripPrefix("/example/", api))
    http.ListenAndServe(":8080", nil)
}
