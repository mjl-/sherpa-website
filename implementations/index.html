Implementations

It's easy to create a sherpa API. Using an existing library makes it even easier:

  • Go server and client: bitbucket.org/mjl/sherpa
  • JavaScript client: github.com/mjl-/sherpajs
  • TypeScript client: github.com/mjl-/sherpats
  • Java server (servlet): github.com/irias/sherpa-servlet (no type information in its sherpadoc)

No implementation for your language?
